## Custom Settings

### Turning Off Compact Folders

Compact Folders feature in VS Code combines single-child folders in the explorer into one compact folder. While this can save space, I prefer to have each folder explicitly listed for easier navigation. Here's how I turned it off:

1. **Open** the VS Code
2. **Navigate** to `File -> Preferences -> Settings -> Features -> Explorer`
3. **Untick** the `Compact Folders` checkbox
